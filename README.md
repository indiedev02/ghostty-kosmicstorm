## Usage

Set the following in your
[Ghostty configuration file](https://ghostty.org/docs/config#file-location):

```ini
theme = kosmicstorm-void.conf
```

### Direct / Manual Installation

1. Download `kosmicstorm-void.conf` from the `themes/` directory.
2. Place it in your Ghostty themes directory:

```
~/.config/ghostty/themes/
```

3. Set the theme in your Ghostty config file:

```ini
theme = kosmicstorm-void.conf
```

4. Reload or restart Ghostty.

> [!NOTE]
> See the Ghostty theme reference for more details:  
> https://ghostty.org/docs/config/reference#theme

## Palette Philosophy

- Dark cosmic purple background
- High-saturation neon accents
- Strong contrast for long coding sessions
- Optimized for truecolor terminals

Kosmicstorm is designed to feel energetic, clean, and immersive without sacrificing readability.
